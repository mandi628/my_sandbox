Default Settings in Typography

"Bootstrap has automatic heading stylings for the fonts and headings. The global default font-size is 14px, with a line-height of 1.428. This is automatically applied to the <body> and paragraph (<p>) elements. All paragraph elements also have a bottom margin equaling half their computed line-height.

Heading 6 = 12px secondary text

Bootstrap Tables

Basic Table

By adding the .table class to the <table> element, you'll get the following basic styling.

Firstname Lastname Email
John Doe john@example.com
Mary Moe mary@example.com
July Dooley july@example.com

The same table, styled by adding the .table-striped class, adds zebra-stripes.

Firstname Lastname Email
John Doe john@example.com
Mary Moe mary@example.com
July Dooley july@example.com

The .table-bordered class, in place of stripes, adds borders to the same table.

Firstname Lastname Email
John Doe john@example.com
Mary Moe mary@example.com
July Dooley july@example.com

And the .table-hover class gives you a gray style when the mouse is hovering.

Firstname Lastname Email
John Doe john@example.com
Mary Moe mary@example.com
July Dooley july@example.com

A .table-condensed class cuts the cell padding in half, giving you a more condensed table display.

Firstname Lastname Email
John Doe john@example.com
Mary Moe mary@example.com
July Dooley july@example.com

You can also use the contextual color classes in the <tr> and <td> elements. They are success (green), danger (red), info (blue), warning (yellow), and active (light gray).

By using .table-responsive class in the <div> surrounding a <table> element, you get a table that will resize to the device width (under 768px wide).

For a complete reference, see the w3schools Bootstrap Tables Reference.
